# weapp-motor-movies

使用官方小程序MINA框架开发的一款电影小程序，旨在学习、研究小程序生态。

## 说明

仿猫眼小程序风格开发的一款电影小程序助手
数据来源：[douban-api-docs](https://github.com/zce/douban-api-docs)

## 功能

- 院线
  - 热映电影
  - 即将上映
- 豆瓣榜单
  - 新片排行榜
  - 豆瓣Top250
- 我的
