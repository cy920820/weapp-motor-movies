# weapp-motor-movies

使用官方小程序MINA框架开发的一款电影小程序

## 项目背景

之前或多或少接触小程序，一直没抽出时间阅读官方文档，发现小程序发展还是很快的，该项目旨在记录、研究小程序生态的过程。

## 说明

仿猫眼小程序风格开发的一款电影小程序助手
数据来源：[douban-api-docs](https://github.com/zce/douban-api-docs)

## 功能

- 院线
  - 热映电影
  - 即将上映
- 豆瓣榜单
  - 新片排行榜
  - 豆瓣Top250
- 我的
